Comment trouver les IDs ?
• Sheets Compta : Créez un nouveau Google Sheets, nommez-le "Arche_Compta", copiez l'ID dans l'URL
• Drive Factures : Ouvrez le dossier Comptabilite/Factures dans Drive, copiez l'ID dans l'URL
• Client ID : console.cloud.google.com → Identifiants → votre client OAuth 2.0
• Webhook : Ouvrez votre Google Sheets → Extensions → Apps Script → Déployer → Gérer les déploiements → URL
